Retexp : projet SQL : création d'une base de donnée opérationnelle
Cahier des charges :

  • - Analyser la demande client et ses besoins en vue de la base de donnée qu'il attend.

  • - Créer un MCD (modèle conceptuel de base de données) de la base de données.

  • - Générer son MLD (modèle logique de la base de données).

  • - Créer la base de donnée elle-même en générant le script de création de la base de données (et en corrigeant si nécessaire les erreurs).

  • - Commenter et expliquer chaque partie de la création de la base de données.

  • - Écrire un script de remplissage de la base de données.

    • Outils utilisés :

  • - WinDesign pour la création du MCD et la génération du MLD puis du script.

  • - MS SQL Server pour la création de la base de données elle-même.

    • Compétences aquises :

  • - TPratique de la création de schémas de bases de données à l'aide d'une demande client.

  • - Pratique du language SQL : création et remplissage de la base de données.
